Channels

Once your AI bot is developed, the next step is to deploy it to a external platforms. Build once and deploy anywhere. Each channel has unique features. Below is a comprehensive list of supported channels:

  1. SocketIO
  2. WhatsApp (via Twilio)
  3. Facebook
  4. Telegram
  5. Slack
  6. Google Hangout
  7. Microsoft Bot Framework

By default, the REST channel is enabled, which supports unidirectional communication. This allows for one-on-one conversations with the bot.

The REST channel provides a REST endpoint that accepts messages and responds with the bot's replies.

SocketIO

The SocketIO channel utilizes web sockets for real-time communication.

To enable this channel, provide the following details:

OptionDescription
bot_message_evtThe event name used by CogniAssist to send messages over SocketIO.
user_message_evtThe event name used by CogniAssist to receive messages over SocketIO.

Enter the required details and toggle the icon to enable the channel.

WhatsApp (via Twilio)

Deploy your AI bot to WhatsApp via Twilio, making it accessible through text messages. Ensure you have a WhatsApp business profile.

To enable this channel, provide the following details:

OptionDescription
account_sidThe account security identifier.
auth_tokenThe authorization token.
twilio_numberThe Twilio phone number.

Enter the required details and toggle the icon to enable the channel.

Facebook

Deploy your AI bot on a Facebook page. First, set up a Facebook page.

To enable this channel, provide the following details:

OptionDescription
verifyThe verification code.
secretThe secret code.
page_access_tokenThe page access token.

Enter the required details and toggle the icon to enable the channel.

Telegram

To deploy your AI bot on Telegram, create a Telegram bot using the Telegram credentials.

To enable this channel, provide the following details:

OptionDescription
access_tokenThe access token.
verifyThe verification code.
webhook_urlThe webhook URL.

Enter the required details and toggle the icon to enable the channel.

Slack

Deploy your AI bot on Slack by creating a Slack app.

To enable this channel, provide the following detail:

OptionDescription
slack_tokenThe Slack token.

Enter the required detail and toggle the icon to enable the channel.

Microsoft Bot Framework

Deploy your AI bot on the Microsoft Bot Framework by creating a Microsoft app.

To enable this channel, provide the following details:

OptionDescription
MICROSOFT_APP_IDThe Microsoft application ID.
MICROSOFT_APP_PASSWORDThe Microsoft application password.

Enter the required details and toggle the icon to enable the channel.
