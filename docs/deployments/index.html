Deployments

Once you have created and tested an AI bot locally, you can deploy it, allowing users to start a direct conversation with the AI bot.

Click Deployments to navigate to the Deployments screen, where you can launch, start, stop, and view execution logs.

Click the Add icon to add multiple deployments.

Launch

Click Launch to launch a fresh instance. Launching a new instance will delete the existing instance in the system.

Start

Click Start to start the launched instance.

Stop

Click Stop to stop the launched instance.

Logs

Click Logs to view execution logs.
